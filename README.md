A premium e-commerce website for luxury shirts with smooth animations, cart functionality, order management, and PDF receipt generation.

## ✨ Features

### 🛍️ Shopping Experience
- **Interactive Product Carousels** - Infinite scrolling collections and bestsellers sections
- **Add to Cart** - Seamless product addition with quantity management
- **Cart Sidebar** - Slide-out cart with item management and real-time updates
- **Promo Codes** - Discount system (e.g., "DISCOUNT50" for 50% off)

### 💳 Checkout System
- **Multi-step Checkout** - Customer info, order summary, and payment selection
- **Payment Options** - COD, Bank Transfer, Credit Card, E-Wallet
- **Form Validation** - Real-time validation for all customer fields
- **Order Summary** - Detailed breakdown with taxes and discounts

### 📋 Order Management
- **Order History** - View all past orders
- **PDF Receipts** - Download professional receipts for any order
- **Local Storage** - Cart and order persistence between sessions

### 🎨 Visual Effects
- **Particle Animation** - Dynamic background with floating gold particles
- **Smooth Scrolling** - Lenis-powered smooth scroll experience
- **GSAP Animations** - Scroll-triggered animations and transitions
- **Responsive Design** - Fully responsive on mobile, tablet, and desktop
